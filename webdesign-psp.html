PRODUSER

Descrição do projeto

Este projeto consistiu em desenvolvimento de web design.

Este é um sistema web para ordem de serviço.

O objetivo deste sistema é ser usado por comerciantes ou pessoas que vendem seus serviços, nele o usuário pode criar um login como administrador e como funcionário, assim algumas áreas ficam restritas para apenas os administradores. Este sistema permite cadastros de clientes, serviços, produtos, gera relatórios financeiros e de vendas.

Este é um sistema web que pode ser acessado de Smartphones, Tablets ou por Desktop ele se ajusta para todos tamanhos de tela
